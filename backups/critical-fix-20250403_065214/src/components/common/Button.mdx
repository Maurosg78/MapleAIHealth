# Button

El componente Button es un elemento interactivo que permite a los usuarios realizar acciones.

## Uso

```tsx
import { Button } from '@/components/common/Button';

// Botón básico
<Button>Click me</Button>

// Botón con variante
<Button variant="secondary">Click me</Button>

// Botón con tamaño
<Button size="lg">Click me</Button>

// Botón con iconos
<Button leftIcon={<Icon />} rightIcon={<Icon />}>
    Click me
</Button>

// Botón en estado de carga
<Button isLoading>Click me</Button>
```

## Props

| Prop      | Tipo                                                       | Por defecto | Descripción                              |
| --------- | ---------------------------------------------------------- | ----------- | ---------------------------------------- |
| variant   | 'primary' \| 'secondary' \| 'outline' \| 'ghost' \| 'link' | 'primary'   | La variante visual del botón             |
| size      | 'default' \| 'sm' \| 'lg' \| 'icon'                        | 'default'   | El tamaño del botón                      |
| isLoading | boolean                                                    | false       | Si el botón está en estado de carga      |
| leftIcon  | ReactNode                                                  | undefined   | Icono a mostrar a la izquierda del texto |
| rightIcon | ReactNode                                                  | undefined   | Icono a mostrar a la derecha del texto   |
| disabled  | boolean                                                    | false       | Si el botón está deshabilitado           |

## Variantes

### Primary

El estilo principal del botón, ideal para acciones principales.

```tsx
<Button variant="primary">Primary Button</Button>
```

### Secondary

Un estilo alternativo para acciones secundarias.

```tsx
<Button variant="secondary">Secondary Button</Button>
```

### Outline

Un botón con borde y fondo transparente.

```tsx
<Button variant="outline">Outline Button</Button>
```

### Ghost

Un botón sin fondo ni borde, solo texto.

```tsx
<Button variant="ghost">Ghost Button</Button>
```

### Link

Un botón que parece un enlace.

```tsx
<Button variant="link">Link Button</Button>
```

## Tamaños

### Default

El tamaño estándar del botón.

```tsx
<Button size="default">Default Button</Button>
```

### Small

Un botón más compacto.

```tsx
<Button size="sm">Small Button</Button>
```

### Large

Un botón más grande.

```tsx
<Button size="lg">Large Button</Button>
```

### Icon

Un botón cuadrado para iconos.

```tsx
<Button size="icon">
    <Icon />
</Button>
```

## Estados

### Loading

El botón muestra un spinner y se deshabilita.

```tsx
<Button isLoading>Loading Button</Button>
```

### Disabled

El botón se deshabilita y no responde a eventos.

```tsx
<Button disabled>Disabled Button</Button>
```

## Iconos

### Left Icon

Agrega un icono a la izquierda del texto.

```tsx
<Button leftIcon={<Icon />}>Button with Left Icon</Button>
```

### Right Icon

Agrega un icono a la derecha del texto.

```tsx
<Button rightIcon={<Icon />}>Button with Right Icon</Button>
```

## Accesibilidad

El botón implementa las siguientes características de accesibilidad:

- Usa el elemento `<button>` nativo
- Soporta estados deshabilitados
- Incluye focus visible
- Mantiene contraste de colores adecuado
- Soporta navegación por teclado
